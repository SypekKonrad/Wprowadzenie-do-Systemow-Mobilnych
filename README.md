# Wprowadzenie do Systemów Mobilnych

Repozytorium zawiera materiały, projekty i zadania realizowane w ramach przedmiotu Wprowadzenie do Systemów Mobilnych na kierunku Informatyka.

## Cel repozytorium

Celem repozytorium jest gromadzenie rozwiązań laboratoriów, projektów oraz materiałów związanych z tworzeniem aplikacji mobilnych.
